## Description

Nestjs State Machine module for [Nest](https://github.com/nestjs/nest) built on top of the [nestjs/event-emitter](https://github.com/nestjs/event-emitter) package.

## Installation

```bash
$ npm i --save @depthlabs/nestjs-state-machine
```
